Project Notes

Future scope: where AthletesPulse can go next.

This FY project focuses on the front-end experience – forms, interactions and structured information. The same interface can later be connected to:

  • AI video analysis – using pose estimation to assess sprinting, jumping and shots.
  • Data-backed injury prediction – using real athlete datasets instead of simple rules.
  • Real scholarship databases – integration with official government and college portals.
  • GIS sports maps – live district-level facility mapping for planning new grounds.

For now, AthletesPulse stays lightweight and easy to run in a browser with only HTML, CSS and JavaScript.
